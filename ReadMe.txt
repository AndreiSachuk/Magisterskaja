Файл описания работы базовой станции

После подключения питания поднимается AP станция 
После старта на дисплее отображенается название сети WiFi и IP адрес для подключения к WEB странице

После подключения к сети необходимо подключиться к сети, перейти по заданному адресу, и выбрать режим работы

Доступные режимы работы:
1) 	Ручное управление
    -	управление при помощи регулировочных резисторов
		- управление при помощи сенсора цвета
2)	Управление при помощи DMX консоли
3)	Упраление через веб страницу

Управление при помощи регулировочных резисторов: в системе присутствует 4 регулировочных резистора: R G B S.
R - регулирует интенсивность красного канала адресного светодиода
G - регулирует интенсивность зеленого канала адресного светодиода
B - регулирует интенсивность зеленого канала адресного светодиода
S - регулирует интенсивность одноцветной ленты, подключенной к системе

Управление при помощи сенсора цвета: управление адресной лентой
К датчику подносится объект для чтения цвета и передает полученные данные по радиоканалу на приемники

Управление при помощи DMX консоли:
При переходе на страницу настройки параметров DMX консоли необходимо указать 4 канала, с которых будет происходить чтение значений R- G- B- S-каналов светодиодов.
R - регулирует интенсивность красного канала адресного светодиода
G - регулирует интенсивность зеленого канала адресного светодиода
B - регулирует интенсивность зеленого канала адресного светодиода
S - регулирует интенсивность одноцветной ленты, подключенной к системе

Упраление через веб страницу:
Присутствует возможность выбора цвета на палитре цветов, так жеприсутствуют ползунки для регулирования значений R- G- B- S-каналов светодиодов.
